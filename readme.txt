Command line utility interface for fetching and displaying the info on matches
that are happening/are scheduled to happen for a specific league of legends region/split

The class now fetches, and updates the JSON data whenever called, providing a seamless
interface.

This branch was made to re-fit the program to read and operate with playoff matches,
i.e., a single best-of-three or best-of-five match per day. Once integrated, the program
should understand the difference between regular-season and playoffs automatically.
