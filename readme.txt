Command line utility interface for fetching and displaying the info on matches
that are happening/are scheduled to happen for a specific league of legends region/split

The class now fetches, and updates the JSON data whenever called, providing a seamless
interface.

This branch was created to test the different ways in which a "league selector" could be
implemented, so we could, e.g. pick to see LCK matches instead of LEC (default) matches.
